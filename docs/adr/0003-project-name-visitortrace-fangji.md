# Name the project VisitorTrace (访迹)

Status: accepted

The project is named VisitorTrace in English and 访迹 in Chinese. The English name covers Pageview Records, geographic observations, and future analytics without limiting the product to map rendering; the Chinese name is a concise semantic counterpart. Repositories, binaries, services, and operational commands use the lowercase identifier `visitortrace`, while Public Map and Pageview remain domain terms rather than brand names.
